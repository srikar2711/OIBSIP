# Number Guessing Game

## Description
A console-based Java game where the player guesses a randomly generated number.

## Features
- Random number generation
- Multiple rounds
- Score calculation
- Hints system
- Menu-driven interface

## Technologies Used
- Java
- Random class
- Scanner class
- Loops and Methods

## How to Run
1. Compile the program:
   javac GuessChallenge.java

2. Run the program:
   java GuessChallenge
